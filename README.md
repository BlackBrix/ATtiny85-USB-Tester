# USB Power Tester based on ATtiny45/85
Simple USB Power Tester based on ATtiny45/85 and INA 219. You can switch between different screens by pressing the SET button.

|Parameter|Value|
|-|-|
|Voltage|3V - 12V|
|Current|max 5A|
|Precision|4mV / 1mA|
